# BashRC Cool
Custom your `~/.bashrc` with BashRC Cool.

### Themes
1. Rose
2. Horse
3. Minion

### Example
![alt text](https://i.postimg.cc/mr0bx176/Selection-691.png)

### Installation:
```
git clone https://github.com/panophan/bashrc-cool.git
cd bashrc-cool
chmod +x setup.bash
sudo ./setup.bash
```

### Command to Configuration Menu
```
bashrc-cool
```
