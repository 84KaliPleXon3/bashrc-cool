# BashRC Cool
Custom your `~/.bashrc` with BashRC Cool.

### Themes
1. Rose
2. Horse

![alt text](https://i.postimg.cc/mr0bx176/Selection-691.png)

### Installation:
```
sudo -i / su
wget https://raw.githubusercontent.com/novranfaathir/bashrc-cool/master/setup.bash
chmod +x setup.bash
./setup.bash
```

### Command to Configuration Menu
```
bashrc-cool
```
